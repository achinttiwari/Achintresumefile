# Achint's Portfolio: Full-Stack Developer

A high-performance personal portfolio website built to showcase UI/UX knowledge, technical expertise, and advanced front-end animations.

## Features
* **Advanced Scroll Animation:** Features an Apple-style custom `ScrollSequence` component utilizing GSAP ScrollTrigger and an HTML `<canvas>` to render a smooth, scroll-linked 300-frame image sequence.
* **Cognitive UI/UX Design:** Layout structured around the Gutenberg diagram (Z-pattern) to naturally guide the user's eye toward key projects and the primary Call-to-Action.
* **Accessible Color Theory:** Implements the 60-30-10 color rule (Cream background, Navy Blue text, Dark Maroon accents) to ensure WCAG 4.5:1 compliant contrast ratios.

## Tech Stack
* **Framework:** Next.js (App Router)
* **Animation:** GSAP & ScrollTrigger
* **Deployment:** Vercel (Edge Network)
* **Repository Management:** Git LFS (Large File Storage) for optimized media delivery

## Installation
1. Clone the repository: 
   `git clone https://github.com/achinttiwari/Achintresumefile.git`
2. Install dependencies: 
   `npm install`
3. If working with the animation frames, ensure Git LFS is installed:
   `git lfs pull`

## Usage
Run the development server with `npm run dev` and open `http://localhost:3000` in your browser.
